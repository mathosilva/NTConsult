# Automação de testes

Projeto de automação, desenvolvido utilizando a linguagem javascript, e usando como ferramento o webdriverio, que é uma biblioteca do nodeJS, e a forma de escrita dos testes foi utilizando o padrão jasmine.

## Pré-requisitos

google Chrome: v112.0
Node: v16.16.0

## Instalação

1- Abra o console de sua preferência
2- Acesse araiz do projeto utilizando o comando "cd"
3- Na raiz do projeto, digite npm install

## Uso

Na Raiz do projeto, digite o comando "npm run wdio"
Após a execução dos testes o resultado será exibido no próprio console
